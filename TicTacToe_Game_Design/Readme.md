1. Design a game of tic-tac-toe.
2. Game has 2 players and a board of size n x n.
3. Provide a way to launch the game.
4. Players should be able to make moves turn-by-turn.
5. If there is a winner or there are no moves left then end the game.
6. One of the two players can be a computer player.
7. Computer player difficulty can be EASY, MEDIUM, HARD.
8. A persistent leaderboard.
